# 🧠 NeuroLeap — AI-Powered EdTech Platform

> *Learn Faster. Think Deeper.*

A fully responsive, multi-page EdTech website built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies, no build step. Designed with a bold dark-neon aesthetic inspired by modern SaaS products.

---

## 🌐 Live Preview

Open `index.html` in any browser — works offline, zero setup required.

## ✨ Features

- **Custom animated cursor** — dot + trailing ring, hover-reactive
- **Scroll-reveal animations** — elements fade up as you scroll
- **Live course filter** — JavaScript category filtering, no page reload
- **Animated stat counters** — numbers count up on viewport entry
- **Tabbed course detail** — Overview / Curriculum / Instructor / Reviews
- **FAQ accordion** — smooth open/close transitions
- **Sticky course sidebar** — enroll card follows scroll on desktop
- **Marquee ticker strip** — infinite scrolling text band
- **Animated SVG rings** — rotating orbit rings on hero visual
- **Floating UI chips** — animated pill badges on hero
- **CSS grid layout** — responsive across all pages
- **Mobile responsive** — hamburger-ready, single-column on small screens
- **Active nav highlighting** — current page link auto-highlighted
- **Anchor-linked sections** — services and footer links resolve to `#section-id`

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/yourusername/neuroleap.git

# Open in browser
cd neuroleap
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

No `npm install`. No build process. Just open and go.

## 🌍 Deployment

Works on any static hosting platform:

| Platform | Steps |
|----------|-------|
| **GitHub Pages** | Push to repo → Settings → Pages → Deploy from `main` branch |
| **Netlify** | Drag & drop the `neuroleap/` folder at netlify.com/drop |
| **Vercel** | `vercel --prod` from project root |
| **Cloudflare Pages** | Connect repo, set build output to `/` |

## 🧩 Tech Stack

- **HTML5** — semantic structure throughout
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **Vanilla JavaScript** — cursor, scroll reveal, counters, tabs, filter, accordion
- **Google Fonts** — Bebas Neue, Space Grotesk, DM Mono (CDN)
- **Zero dependencies** — no React, no Tailwind, no jQuery
